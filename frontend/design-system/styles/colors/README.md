# 配色系统使用指南

## 主题特点

### 1. Vibrant（充满活力）
- 主色调：明亮的绿色，代表健康和活力
- 辅助色：清新的蓝色，提供视觉平衡
- 强调色：活力橙、梦幻紫、清新青

### 2. Soft（温暖舒适）
- 主色调：柔和的紫色，营造温馨氛围
- 辅助色：温暖的橙色，增添亲和力
- 强调色：薄荷绿、桃粉色、淡紫色

### 3. Professional（稳重可靠）
- 主色调：沉稳的蓝色，体现专业性
- 辅助色：可靠的绿色，增强信任感
- 强调色：靛蓝、青色、琥珀色

## 使用建议

1. 主色调应用
   - 主要按钮和重要操作
   - 品牌标识和关键信息
   - 导航和标题栏

2. 辅助色应用
   - 次要按钮和操作
   - 背景和装饰元素
   - 分类和标签

3. 强调色应用
   - 特殊功能突出
   - 数据可视化
   - 状态指示

4. 功能色应用
   - 成功/警告/错误提示
   - 操作反馈
   - 状态展示
